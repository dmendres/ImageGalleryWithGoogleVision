Using Google Vision

Google Vision API is able to detect and classify many kinds of objects within images. Check the boxes for features you wish to locate, then click on an image in the Image Gallery. Google Vision will be activated to analyze the image for the features selected. Information will be displayed below. Note: not all of these images have analyzable content.
